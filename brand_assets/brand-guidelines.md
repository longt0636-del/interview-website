# LongIELTS Brand Guidelines

## Colour Palette

| Name        | Hex       | Usage                                          |
|-------------|-----------|------------------------------------------------|
| Navy        | `#0B3D5C` | Hero backgrounds, headings, primary dark text  |
| Teal        | `#1D9E75` | CTA buttons, badges, icon marks, labels        |
| Teal light  | `#5DCAA5` | Taglines on dark bg, secondary accents         |
| Mint bg     | `#E1F5EE` | Info callouts, light section backgrounds       |
| Amber       | `#EF9F27` | Score badges, stat numbers — NEVER body text bg|
| Gold light  | `#FAC775` | Secondary amber accent                         |
| Ink         | `#2C2C2A` | All body copy on white/light backgrounds       |
| Off-white   | `#F4F8F5` | Page background, body cards, brochure bg       |

## Text on Background Pairing Rules

| Background  | Text colour | Use for                               |
|-------------|-------------|---------------------------------------|
| Navy        | White       | Hero headings, dark sections          |
| Teal        | White       | CTA buttons, badges                   |
| Mint        | Navy        | Info callouts, light cards            |
| Off-white   | Ink         | Body cards, page background           |
| Amber       | Dark brown  | Score badges ONLY                     |
| White       | Ink / Navy  | Primary body copy, most UI elements   |

## Typography

### Fonts (all free on Google Fonts)
- **Serif** — Playfair Display (preferred) or Lora: `font-serif` class
- **Sans** — Inter (preferred) or Poppins: `font-sans` class
- **Mono** — DM Mono: `font-mono` class

### Where to use each font
- `font-serif` → Logo wordmark, H1–H2 headings, slide titles, pull quotes
- `font-sans` → Body copy, buttons, labels, badges, nav items
- `font-mono` → Band scores, statistics, data tables, score badges

### Type Scale
| Style       | Font       | Size / Weight  | Usage                          |
|-------------|------------|----------------|--------------------------------|
| Display     | Serif Bold | 32pt / 700     | Hero heading                   |
| Heading 1   | Serif Bold | 24pt / 700     | Section title                  |
| Heading 2   | Serif Bold | 18pt / 700     | Sub-section                    |
| Heading 3   | Sans Bold  | 14pt / 700     | Card title                     |
| Body large  | Sans       | 11pt / 400     | Primary body                   |
| Body small  | Sans       | 9pt / 400      | Captions, footnotes            |
| Label       | Sans Med   | 8pt / 500      | Tags, badges, nav items        |
| Score hero  | Mono Bold  | 36pt / 700     | Band score display             |
| Score       | Mono Bold  | 18pt / 700     | Score in tables                |
| Data label  | Mono       | 9pt / 400      | Table numbers, stats           |

## Voice & Tone

**DO:** "100% học viên tăng ít nhất 0.5 band." — factual, calm, specific
**DO:** "Lớp tối đa 8 người — thầy biết tên, biết điểm yếu từng em."
**DON'T:** "Cam kết đầu ra 100% TUYỆT ĐỐI!!!" — hype, caps, exclamation overload
**DON'T:** "Phương pháp độc quyền siêu đỉnh!" — vague superlatives
